# bupt-is-ta-recruitment-system
Agile-based TA Recruitment System for BUPT International School.

# Group Collaboration Rules

## Branch Rule
Do not push directly to `main`.
Each member must create their own branch.

Branch name format:
`name/feature`

Examples:
- `cindy/prototype-home`
- `ruizhe/backlog-user-stories`

## Workflow
1. Pull the latest `main`
2. Create a new branch
3. Work on your own branch
4. Commit and push your branch
5. Open a Pull Request to `main`
6. Another group member or group leader must approve the PR
7. Merge after approval

## Important
- Do not fork the repository
- Each member must have their own commits
- Each member must create at least one PR before 22 March 2026
